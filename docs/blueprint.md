# **App Name**: HyipNexus

## Core Features:

- Investment Plan Display: Showcase various HYIP investment plans with detailed descriptions of returns, risks, and durations.
- User Authentication: Secure user accounts with login, registration, and password management.
- Investment Tracking: Enable users to monitor their investments, track returns, and view transaction history in real-time, utilizing the Supabase database.
- Payment Gateway Integration: Integrate secure payment gateways (like Stripe) to facilitate deposits and withdrawals.
- Referral Program: Implement a referral system that rewards users for inviting new investors, managed through Supabase.
- Admin Panel: Provide an admin interface for managing investment plans, user accounts, and overall system settings, stored in Supabase.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to inspire trust and security.
- Background color: Light gray (#ECEFF1), almost white, for a clean and modern aesthetic.
- Accent color: Yellow-orange (#FFAB40) to highlight key interactive elements.
- Body and headline font: 'Inter', a versatile sans-serif.
- Use clear, professional icons to represent different investment features and options.
- A clean, structured layout with a focus on displaying data clearly and concisely.
- Subtle transitions and loading animations to create a smooth user experience.
